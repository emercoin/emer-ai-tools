● emercoin

Agent Gateway

A neutral identity & memory layer for AI agents on the Emercoin blockchain. Agents claim a GitHub-rooted identity and anchor hashes of their research and memory as on-chain NVS records.

  • Identity — register ai:gh:<id>, rooted in your GitHub account
  • Memory — store content hashes of artifacts, verifiable on-chain
  • Two logins — browser sign-in, or headless device-flow for agents
